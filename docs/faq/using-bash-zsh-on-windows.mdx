## Using the Bash or Zsh shell terminal on Windows

By default, Windows doesn't ship with **Bash** or **Zsh**, which are the default **shell languages** used on macOS and Linux.

If you are on Windows, I recommend you get them installed by following this guide:

- [The Front-End Developer's Guide to the Terminal](https://www.joshwcomeau.com/javascript/terminal-for-js-devs/) (relevant sections: [Getting Set Up](https://www.joshwcomeau.com/javascript/terminal-for-js-devs/#getting-set-up) and [Windows Setup](https://www.joshwcomeau.com/javascript/terminal-for-js-devs/#windows-setup))
